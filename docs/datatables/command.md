## Text
Set command text.
```csharp
.Text("example")
```

## OnClick
Set onClick function name.
```csharp
.OnClick("example")
```

## IconClass
Set icon css class.
```csharp
.IconClass("example")
```

## BtnClass
Set button css class.
```csharp
.BtnClass("example")
```

## Width
Set percentage of column where the command is.
```csharp
.Width("example")
```
