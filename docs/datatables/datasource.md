## URL
Set data source URL.
```csharp
.URL("example")
```

## Method
Set Ajax Http method.
```csharp
.Method("example")
```

## Data
Passing additional data to action set name of javascript function.
```csharp
.Data("example")
```

## Naming
Set naming convention strategy. If not set uses default.
```csharp
.Naming(default)
```
