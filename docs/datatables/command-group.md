## Text
Set command text.
```csharp
.Text("example")
```

## IconClass
Set icon css class.
```csharp
.IconClass("example")
```

## BtnClass
Set button css class.
```csharp
.BtnClass("example")
```

## Width
Set percentage of column where the command is.
```csharp
.Width("example")
```

## Item
Add a Command Group item.
```csharp
.Item("example", "example")
```
